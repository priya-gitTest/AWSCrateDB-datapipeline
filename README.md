# AWSCrateDB-datapipeline
Real-time Climate Data Pipeline on AWS with CrateDB and Graphana

✅ Streaming data ingestion using Apache Kafka (MSK)

✅ Serverless data processing with AWS Lambda

✅ Distributed SQL database (CrateDB) for time-series data

✅ Real-time visualization with Grafana dashboards

✅ Event-driven architecture with MSK triggers

✅ Cloud infrastructure (AWS VPC, EC2, S3, CloudFormation)

✅ Python data engineering (producer/consumer pattern)

✅ Data modeling (GeoPoint, dynamic objects, time-series)

Ref : https://github.com/crate/realtime-demo
**Note**: This project was developed as part of a hands-on AWS workshop to 
demonstrate real-world streaming data engineering skills on 03-12-2025.
